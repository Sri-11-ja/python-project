# python-project
STUDENT SCORE ANALYSIS USING LINEAR REGRESSION

Project Overview:
       This project predicts student exam scores based on various factors such as study hours, sleep hours, attendance, and previous scores using Linear Regression in Python.
       The goal is to understand how different features affect student performance and build a predictive model.

Dataset Description:

The dataset contains the following features:

hours_studied  – Number of hours spent studying
sleep_hours  – Daily sleep duration
attendance_percent  – Attendance percentage
previous_scores – Past academic performance
exam_score  – Target variable 

Machine Learning Model:
Linear Regression:
   Used for predicting continuous numerical values
   Suitable for this dataset since the target is numeric

Results:
        The model successfully predicts exam scores based on input features
        Visualization shows the relationship between actual and predicted values
